# Local Automaton Buyer Site

Upload this folder as a static site. The entrypoint is `index.html`.

## Vercel Publish

```sh
vercel login
vercel --yes --prod
```

## Purpose

Sell the 19 USDC Local Automaton setup package and route buyers to intake/payment details.

## Payment Safety

The page contains only a public receiving address. It does not contain private keys, seed phrases, passwords, or API secrets.
